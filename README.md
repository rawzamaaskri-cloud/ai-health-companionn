# ai-health-companionn

AISANTÉ is a premium intelligent digital health platform built for Algeria, connecting patients, doctors, and pharmacies in an AI-powered, secure, and fully integrated ecosystem.

## Key Features
- **Patient Workspace**: Digitized health records, online appointments, medication trackers, and IoT integration.
- **Doctor Workspace**: Patient monitoring dashboard, electronic prescriptions, and AI clinical assistant tools.
- **Pharmacy Workspace**: Real-time e-prescription receipt, inventory lookup, and chat channels.
- **Connected Smart Bracelet**: Streams vitals (heart rate, activity tracker) to the medical portal with real-time anomaly alerts.
- **Multi-lingual support**: Full support for French, English, and Arabic (complete layout RTL transformation).
